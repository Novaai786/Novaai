📘 Email Campaign Setup Guide (Google Apps Script)

Is guide se aap apne Gmail account se free mein roazana 500 emails tak bhej sakte hain — seedha NovaAICRM se, bina kisi paid tool ke. Waqt lagega: 10-15 minutes (sirf ek dafa karna hai).

Step 1: Google Sheet Banao

sheets.google.com par jayein aur Blank Spreadsheet banayein, naam rakhein NovaAICRM Campaigns. Pehli row mein ye headers likhein (A se G tak): Email, Subject, Body, Status, SentAt, OpenedAt, RowID. URL se apni Spreadsheet ID copy kar lein (/d/ aur /edit ke darmiyan wala hissa).

Step 2: Apps Script Editor Kholein

Google Sheet ke andar: Extensions → Apps Script. Default code delete kar dein.

Step 3: Ye Code Paste Karein

// ===== NovaAICRM Email Campaign Script =====

// STEP A: Yahan apni Spreadsheet ID paste karein
const SHEET_ID = 'YAHAN_APNI_SPREADSHEET_ID_PASTE_KAREIN';
const SHEET_NAME = 'Sheet1';

function saveDeploymentUrl() {
  const url = ScriptApp.getService().getUrl();
  PropertiesService.getScriptProperties().setProperty('DEPLOYMENT_URL', url);
  Logger.log('Saved URL: ' + url);
}

function getDeploymentUrl() {
  return PropertiesService.getScriptProperties().getProperty('DEPLOYMENT_URL');
}

function sendPendingEmails() {
  const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName(SHEET_NAME);
  const data = sheet.getDataRange().getValues();
  const deploymentUrl = getDeploymentUrl();

  for (let i = 1; i < data.length; i++) {
    const row = data[i];
    const [email, subject, body, status] = row;
    const rowId = i + 1;

    if (status === 'Pending' && email) {
      try {
        const trackingPixel = deploymentUrl
          ? '<img src="' + deploymentUrl + '?track=' + rowId + '" width="1" height="1" style="display:none;">'
          : '';

        const htmlBody = body.replace(/\n/g, '<br>') + trackingPixel;

        GmailApp.sendEmail(email, subject, body, {
          htmlBody: htmlBody
        });

        sheet.getRange(rowId, 4).setValue('Sent');
        sheet.getRange(rowId, 5).setValue(new Date());
        sheet.getRange(rowId, 7).setValue(rowId);

      } catch (err) {
        sheet.getRange(rowId, 4).setValue('Failed: ' + err.message);
      }

      Utilities.sleep(1000);
    }
  }
}

function doGet(e) {
  const rowId = e.parameter.track;

  if (rowId) {
    try {
      const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName(SHEET_NAME);
      const currentOpenedAt = sheet.getRange(parseInt(rowId), 6).getValue();

      if (!currentOpenedAt) {
        sheet.getRange(parseInt(rowId), 6).setValue(new Date());
      }
    } catch (err) {
      Logger.log('Tracking error: ' + err.message);
    }
  }

  const pixel = Utilities.base64Decode(
    'R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBTAA7'
  );
  return ContentService
    .createBinaryOutput(pixel)
    .setMimeType(ContentService.MimeType.GIF);
}

function createTimeTrigger() {
  const triggers = ScriptApp.getProjectTriggers();
  triggers.forEach(t => {
    if (t.getHandlerFunction() === 'sendPendingEmails') {
      ScriptApp.deleteTrigger(t);
    }
  });

  ScriptApp.newTrigger('sendPendingEmails')
    .timeBased()
    .everyMinutes(10)
    .create();

  Logger.log('Trigger created successfully!');
}

Code mein YAHAN_APNI_SPREADSHEET_ID_PASTE_KAREIN ki jagah apni asal Spreadsheet ID (Step 1 se) paste karna na bhoolein.

Step 4: Deploy Karein (Web App)

Deploy → New deployment → gear icon (⚙️) → Web app select karein.
Execute as: Me  |  Who has access: Anyone
Deploy dabayein, permission allow karein ("Advanced" → "Go to project (unsafe)" pe click karke), aur mila hua Web app URL copy kar lein.

Step 5 & 6: URL Save Karein aur Trigger Chalayein

Function dropdown se saveDeploymentUrl select karke Run ▶️ karein.
Phir createTimeTrigger select karke dobara Run ▶️ karein — ab script har 10 minute mein khud pending emails check kar ke bhejega.

Step 7: CRM Mein Connect Karein

Upar "Sender Setup" mein apna Web App URL paste karein aur Save Settings dabayein. Bas, ab yahan se campaigns bhej sakte hain.

⚠️ Important:
  • Free Gmail limit: 500 emails/day.
  • Ye script sirf aapke apne Gmail account se kaam karta hai — kisi ka bhi email/password nahi maanga jata.
  • Zyada limit ke liye Google Workspace account use karein (1500/day).
